Getting Started
Setting up our plugin is simple. Just choose a method below.

Recommended

Copy and paste the following code into the HTML section where you want our plugin to appear. Organizations often incorporate our plugin into a dedicated matching gifts page, and into the donation process. This is the recommended way to embed our plugin.
<script>var DDCONF = { API_KEY: "{{api_key}}" };</script>  <script src="https://doublethedonation.com/api/js/ddplugin.js"></script> <link href="https://doublethedonation.com/api/css/ddplugin.css" rel="stylesheet"> <div id="dd-container" ng-app="dd.plugin"> <div class="well"> <div ng-include="search_template"></div> <p><center><a href="https://doublethedonation.com/matching-grant-resources/matching-gift-basics/">Matching Gift</a> and <a href="https://doublethedonation.com/matching-grant-resources/volunteer-grant-basics/">Volunteer Grant</a> information provided by <br> <a href="https://doublethedonation.com"><img alt="Powered by Double the Donation" src="https://doublethedonation.com/api/img/powered-by.png" /></a></center></p> </div> </div>

iFrame

Some software platforms strip Javascript or CSS from user-entered input, causing our plugin to not appear correctly. If you have any issues with the standard code, try using the following iFrame code where you want our plugin to appear.
<iframe src="https://doublethedonation.com/api/iframe/{{api_key}}" height="500" width="100%"></iframe>

Integrations
